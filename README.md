# python 机器学习入门实战
关于传统机器学习算法的实战案例

从简单的回归开始逐步带你开始机器学习之旅，也许你一开始并不懂得算法的原理，但是慢慢的实战之后你会发现机器学习的乐趣。
主要是介绍一些传统的机器学习模型
+ 逻辑回归
+ SVM
+ KNN
+ MLP
+ 贝叶斯
+ 聚类问题
+ 回归问题
